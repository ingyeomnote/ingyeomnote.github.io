안녕하세요! Ingyeom의 기술 블로그에 오신 것을 환영합니다.

## 블로그 소개

이 블로그는 Hugo를 기반으로 구축된 개인 기술 블로그입니다.
개발, 프로젝트, 그리고 다양한 기술에 대한 기록을 공유하는 공간입니다.

## 주요 카테고리

- **개발**: 프로그래밍 언어, 프레임워크, 라이브러리 등
- **프로젝트**: 진행한 프로젝트들의 기록과 회고
- **기술**: 새롭게 배운 기술이나 트러블슈팅 경험

## 블로그 기술 스택

- **Static Site Generator**: Hugo
- **Theme**: PaperMod
- **Hosting**: GitHub Pages
- **CI/CD**: GitHub Actions

앞으로 유익한 기술 콘텐츠로 찾아뵙겠습니다. 감사합니다!
