## Hugo란 무엇인가?

### 실생활 비유로 이해하기

Hugo를 이해하려면 먼저 **정적 사이트**와 **동적 사이트**의 차이를 알아야 합니다.

#### 동적 사이트 (워드프레스 같은)
- **음식점에 비유**: 주문이 들어올 때마다 주방에서 요리를 만듭니다
- 사용자가 페이지를 요청할 때마다 서버가 데이터베이스에서 정보를 가져와서 HTML을 만들어 보여줍니다
- 장점: 실시간 데이터, 사용자 맞춤 콘텐츠
- 단점: 느리고, 서버 비용이 들고, 복잡합니다

#### 정적 사이트 (Hugo로 만든)
- **편의점 도시락에 비유**: 미리 다 만들어서 진열해둡니다
- 미리 모든 HTML 파일을 만들어두고, 요청이 오면 그냥 파일을 보여줍니다
- 장점: 빠르고, 거의 공짜이고, 안전합니다
- 단점: 실시간 업데이트가 어렵습니다 (블로그에는 문제없음!)

### Hugo는 뭐하는 도구인가?

Hugo는 **정적 사이트 생성기(Static Site Generator)**입니다.

```
마크다운 파일(.md) → Hugo → HTML 파일들
```

**예를 들어볼까요?**

당신이 이런 마크다운 파일을 작성했다고 해봅시다:

```markdown
---
title: "내 첫 포스트"
date: 2025-11-04
---

# 안녕하세요!

이것은 내 첫 블로그 글입니다.
```

Hugo는 이걸 읽고, 예쁜 HTML 페이지로 변환해줍니다:
- 헤더, 푸터 추가
- CSS 스타일 적용
- 목록 페이지에도 자동으로 추가
- 태그, 카테고리 페이지도 자동 생성

**왜 Hugo인가?**
- ⚡ **엄청나게 빠릅니다**: Go 언어로 만들어져서 1000개 페이지도 1초 안에 빌드
- 📦 **설치가 쉽습니다**: 단일 실행 파일 하나만 설치하면 끝
- 🎨 **테마가 풍부합니다**: 무료 테마가 수백 개
- 📝 **마크다운으로 글 작성**: 워드프레스처럼 복잡한 에디터 필요 없음

## GitHub Pages란 무엇인가?

### 무료 웹 호스팅 서비스

GitHub Pages는 **GitHub이 제공하는 무료 웹사이트 호스팅 서비스**입니다.

#### 일반 웹 호스팅의 문제
- 💸 웹 호스팅 업체에 돈을 내야 합니다 (월 5,000원~50,000원)
- 🔧 서버 설정, 데이터베이스 관리 등이 복잡합니다
- 📊 트래픽이 많으면 추가 비용이 발생합니다

#### GitHub Pages의 장점
- ✅ **완전 무료**: 트래픽 제한도 넉넉합니다
- ✅ **간단한 배포**: Git push만 하면 자동으로 배포됩니다
- ✅ **HTTPS 자동 지원**: 보안 인증서 무료 제공
- ✅ **CDN 자동 적용**: 전 세계 어디서나 빠르게 접속

### GitHub Pages 작동 원리

```
1. GitHub 저장소 만들기
   ↓
2. HTML 파일들을 저장소에 push
   ↓
3. GitHub이 자동으로 웹사이트로 만들어줌
   ↓
4. https://당신ID.github.io 주소로 접속 가능!
```

**예를 들어:**
- 저장소 이름: `ingyeomnote.github.io`
- 접속 주소: `https://ingyeomnote.github.io`

## Hugo + GitHub Pages 조합의 마법

### 왜 이 둘을 함께 쓰는가?

Hugo와 GitHub Pages는 **천생연분**입니다:

```
[당신]                [GitHub Pages]
   ↓                       ↑
마크다운으로           HTML 파일들을
글 작성                   호스팅
   ↓                       ↑
   → [Hugo로 변환] →
```

#### 전체 흐름

1. **로컬에서 작업** (당신의 컴퓨터)
   ```bash
   hugo new posts/my-post.md  # 새 글 생성
   ```

2. **내용 작성** (마크다운 에디터)
   ```markdown
   ---
   title: "제목"
   ---

   # 내용 작성...
   ```

3. **Hugo로 빌드** (HTML 생성)
   ```bash
   hugo  # public/ 폴더에 HTML 생성
   ```

4. **GitHub에 push** (배포)
   ```bash
   git add .
   git commit -m "새 글 추가"
   git push
   ```

5. **자동 배포 완료!** 🎉
   - 1~2분 후 `https://당신ID.github.io`에 반영됨

## 비용은 얼마나 들까?

### 다른 방법들과 비교

| 방법 | 월 비용 | 난이도 | 속도 |
|------|---------|--------|------|
| **워드프레스** (호스팅) | 5,000~20,000원 | 중 | 느림 |
| **티스토리/네이버** | 무료 | 쉬움 | 보통 |
| **Hugo + GitHub Pages** | **완전 무료** | 중 | **매우 빠름** |
| Medium | 무료~$5 | 쉬움 | 보통 |

### Hugo + GitHub Pages의 장점 정리

✅ **완전 무료**
- 호스팅 비용 0원
- 도메인만 원하면 연간 1만원 정도 (선택사항)

✅ **속도가 매우 빠름**
- 정적 파일이라 로딩이 순식간
- GitHub CDN으로 전 세계 어디서나 빠름

✅ **마크다운으로 글 작성**
- 워드프레스처럼 복잡한 에디터 필요 없음
- Git으로 버전 관리 가능

✅ **커스터마이징 자유도가 높음**
- HTML/CSS 직접 수정 가능
- 테마 무한 수정 가능
